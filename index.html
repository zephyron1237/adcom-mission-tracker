Calculator is in BETA. Be wary of results. (Details/Tips)
Details: The calculator does a basic simulation, normalizing each generator's output rate and running until the goal is met or it reaches 24h. This averages out things like crits and long run speeds, which introduces some error. The calculator currently does not take airdrops or unlocking new generators into account. Please let me know if you find any issues or have a suggestion.
Tips:
  • Enter numbers like 1.52 AA with a space in-between.
  • Leave Speed blank or 0 if not running, use 1 if you are manually running it constantly, 0.5 if you're manually running it half the time, etc.
